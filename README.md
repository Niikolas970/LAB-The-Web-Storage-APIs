# 📱 APP de Pokémon

## 📖 Descripción

APP de Pokémon desarrollada con **HTML, CSS y JavaScript**, que permite buscar Pokémon utilizando la API pública de Pokémon (PokeAPI), visualizar su información y administrar una lista de favoritos almacenada en el navegador mediante Local Storage.

## 🚀 Funcionalidades

* Buscar Pokémon por nombre.
* Mostrar imagen y nombre del Pokémon encontrado.
* Agregar Pokémon a favoritos.
* Evitar favoritos duplicados.
* Almacenar favoritos en Local Storage.
* Mostrar la lista de favoritos al recargar la página.
* Eliminar Pokémon de la lista de favoritos.
* Diseño responsivo y amigable para el usuario.
* Video de fondo temático de Pokémon.

## 🛠️ Tecnologías Utilizadas

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage
* Fetch API
* PokeAPI

## ▶️ Cómo Ejecutar el Proyecto

1. Clonar el repositorio.
2. Abrir la carpeta del proyecto.
3. Ejecutar el archivo `index.html` en el navegador.
4. Buscar un Pokémon por nombre.
5. Agregarlo a favoritos y administrarlo desde la aplicación.
